# Getting Started

Welcome to **Vicky Docs**.

## Configure Admin

1. Open `/admin/login`
2. Sign in with your admin password
3. Open `/admin/settings`
4. Configure your GitHub owner/repo/branch/docs path
5. Save and test connection

## Create a Page

Go to `/editor`, choose or create a page, then save with a commit message.

> [!TIP]
> Use `docs` as your GitHub docs path for the default setup.
